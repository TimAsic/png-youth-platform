# PNG Map Visualization

## Vision

The PNG Map Visualization is a national dashboard showing youth participation, proposals, and issues across Papua New Guinea.

The goal is to transform discussions into visual insights that help users, communities, and decision-makers understand what matters most across the country.

---

# National Pulse Map

The home map displays all provinces.

Each province is colored based on activity levels.

Color System:

- Gray = Low Activity
- Light Blue = Moderate Activity
- Dark Blue = High Activity
- Gold = Leading Province for an Issue

Metrics used:

- Active Users
- Proposal Count
- Support Count
- Discussion Activity

---

# Province Detail View

When a province is selected, users can see:

- Active Users
- Active Proposals
- Top Issues
- Average Proposal Score
- Total Support Count

Example:

Province: Morobe

Active Users: 2143

Active Proposals: 87

Top Issue: Employment

Average Proposal Score: 145

Support Count: 12004

---

# Issue Heatmap

Users can switch between issue views.

Available views:

- Education
- Employment
- Health
- Infrastructure
- Governance
- Environment

The map updates to show where each issue is most discussed.

---

# Proposal Support Distribution

Each proposal displays support by province.

Example:

Youth Apprenticeship Program

NCD: 250
Morobe: 180
Eastern Highlands: 90
Milne Bay: 45
Manus: 15

This helps determine whether support is local or national.

---

# Province Comparison Tool

Users can compare provinces.

Comparison Metrics:

- Users
- Proposals
- Support
- Top Issues
- Average Proposal Score

Purpose:

Identify similarities and differences between provinces.

---

# Future AI Insights

The system will eventually answer questions such as:

- Which issues matter most in each province?
- Which proposals have national support?
- Which provinces are underrepresented?
- Which solutions have the highest potential impact?

The AI system will use proposal data, support data, and provincial statistics to assist decision-making.

---

# Data Requirements

For each province store:

- Province Name
- Population
- Active Users
- Proposal Count
- Support Count
- Average Proposal Score
- Top Issues

This data powers the map visualization and analytics system.
