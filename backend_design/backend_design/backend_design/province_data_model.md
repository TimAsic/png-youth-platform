# Province Data Model

## Purpose

This file defines the province-level data used by the PNG Youth Platform.

The information supports:

- Proposal scoring
- Population weighting
- Map visualization
- AI decision support
- Provincial analytics

---

## Province Record Structure

Each province stores:

- Province Name
- Population
- Active Users
- Proposal Count
- Support Count
- Average Proposal Score
- Top Issues

---

## Example

Province: National Capital District

Population: 450000

Active Users: 2500

Proposal Count: 180

Support Count: 12000

Average Proposal Score: 145

Top Issues:
- Employment
- Housing
- Education

---

## Statistics Calculated

The system calculates:

- Participation Rate
- Proposal Activity Rate
- Support Density
- Provincial Influence Score
